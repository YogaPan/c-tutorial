GNU Compiler Collection(GCC)
===========================

Options:
        -c         Compile and assembly but not link, this will generate *.o files.
        -o:        Place the output input file.
        -w:        Don't show any warnings.
        -Wall:     Show all compile warnings.
        -g:        This will let you use gdb to debug.
        -Os:       Optimize level.
        -ansi:     anci C.
        -S:        Compile only. This will generate *.S files.
        -E:        Preprocessor only.
        -Idir:     Header files seaching path.
        -llibrary: Link library.
        Ldir:      librarys searching path.

Makefile
========

1. Variable:

   If VAR2 change, and VAR1 will change too.

           VAR1 = VAR2

   If VAR2 changes, and VAR1 will keep the origin value.

           VAR1 := VAR2

   VAR1 ?= VAR2
   VAR1 += VAR2
